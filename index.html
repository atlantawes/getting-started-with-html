Interacting + Internet

HTML means Hypertext Markup Language.
HTML documents are the heart of the World Wide Web,
dictating what you see and how you see it.
They are made up of both content and commands,
which combined are referred to as HTML elements.

What I've called commands in the above paragraph
are referred to as tags. Tags are instructions that
we, as programmers, use to tell a browser the type
of element to display on the webpage. They are
fundamental: without them, we would not be able to
even see a single line of text.

There are two types of elements: inline and block.
A single line of text can contain multiple inline elements,
such as my above definition of HTML. Several inline
elements were employed to display bold text,
itallics, and bold itallics. A block element forms an
invisible box around the contents inside.
